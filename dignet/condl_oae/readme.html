Basic pipeline:

  1. run load_sentences.php
  2. run loadPubmedIDs.php
  3. run loadPubmedIDsTKI.php
  4. run loadPubmedIDsTKIdrugs.php
  5. run batchLoadScoresPubmed.php
  6. run runStat.php

Final results are shown at http://ignet.hegroup.org/dignet/condl_oae/listQueries.php.

To export results as zip file, run export.php
